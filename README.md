# Module 1: High-Performance Computing with NumPy

This module covers the core mathematical foundations required for Machine Learning, implemented from scratch using Python's NumPy library.

## 📌 Key Implementations:
1. **Vectorization vs Python Lists:** Benchmarked speed performance showing NumPy computing 25x+ faster than standard loops.
2. **Matrix Multiplication (Dot Product):** Implemented the core engine of Neural Networks ($Z = X \cdot W + b$) handling multi-dimensional weight matrices.
3. **Broadcasting & Boolean Masking:** Efficient data transformation and conditional filtering without explicit looping.
4. **Statistical Operations & Argmax:** Extracting data insights and predicting class indices for classification models.

**Tech Stack:** `Python`, `NumPy`
